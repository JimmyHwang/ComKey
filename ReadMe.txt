-------------------------------------------------------------------------------
 COMKEY Version 1.00
-------------------------------------------------------------------------------
Options - 
  -s x      Specified port numver
  -d        Run with daemon Mode
  -v        Enable version flag

Examples:
  Run with Normal Mode
    sudo ./comkey -s 1 -d

  Run with Daemon Mode
    sudo ./comkey -s 1 -d
  
  Kill Daemon Process
    sudo pkill comkey
